## Getting Started

### Installing

```bash
brew tap polyium/homebrew-taps

brew install polyforge
```

###### Upgrading

```bash
brew update && brew upgrade polyforge
```

###### Reinstalling

```bash
brew update
brew uninstall --cask polyforge
brew install polyforge
```

## Usage

```bash
polyforge --help
```
